Proyecto final Javascript.

Inventario Online.

Mi proyecto es un inventario online que realiza una carga inicial de productos y luego permite ingresar y quitar productos manejando tambien un stock.

Para esta entrega:

1-Utilicé Sweet Alert 2 como librería.
2-Utilicé promesas y async await para crear las alertas al ingresar un producto y tambien para la primera carga.
3-Agregué algunos estilos básicos.
4-Utilicé fetch para cargar los productos iniciales desde un archivo .json.
